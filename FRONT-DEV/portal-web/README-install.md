# **✨ Como executar**

**Para que esse projeto funcione corretamente, é preciso estar com o servidor rodando.**

> - Entre no projeto `cd portal-web`
> - Instale os pacotes com `npm install`.
> - Execute `npm start` para iniciar o servidor.
